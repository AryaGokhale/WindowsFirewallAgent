# WindowsFirewallAgent
ZTNA is the manifestation of the policies designed to implement fine grained control and access to resources. Windows Firewall agent is acts as a policy manager, it is a windows service that queries OPA roll based access control policies to whitelist applications in windows firewall.

Refer the OPA's roll based access contol policies here: https://github.com/AryaGokhale/opa_rbac
